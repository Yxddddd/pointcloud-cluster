## Dependency  
**ROS** (tested with kinetic)
**PCL**

## How to build with catkin    
```
cd ~/catkin_ws/src  
git clone https://github.com/Yxddddd/pointcloud-cluster.git
cd ~/catkin_ws  
catkin_make 
```
